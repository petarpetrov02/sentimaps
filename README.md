# Sentimaps by Petar Petrov

**Sentimaps** is a powerful Streamlit app that maps customer **sentiment**, **emotion**, and **experience categories** from Google or OpenTable review data. It uses AI-powered language processing to classify, extract, and visualise the emotional landscape of hospitality experiences ‚Äî helping restaurants and analysts turn qualitative feedback into actionable insight.
